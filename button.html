<Button />

Buttons communicate the action that will occur when the user touches them

Usage:

const MyComponent = () => (
  <Button raised onPress={() => console.log('Pressed')>}
   Press me
  </Button>
);

Props

disabled: boolean

Disable the button

compact: boolean

Use a compact look, useful for flat buttons in a row

raised: boolean

Add elevation to button, as opposed to default flat appearance

primary: boolean

Use to primary color from theme

dark: boolean

Text color of button, a dark button will render light text and vice-versa

loading: boolean

Whether to show a loading indicator

icon: IconSource

Icon name. Can be a string (name of MaterialIcon), an object {of shape {uri: 'https://path.to'}}, required image from assets (const icon = reqiure('../path/to/image.png')), or any valid React-Native Component (e.g. <Image />)

color: string

Custom text color for flat button, or background color for raised button

children: string | Array<string>

Button text

onPress: Function

Function to execute on press

style: ViewPropTypes.style
*theme: Theme
