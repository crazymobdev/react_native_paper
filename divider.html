<Divider />

A divider is a thin, lightweight rule that groups content in lists and page layouts

Usage:

const MyComponent = () => (
  <View>
    <Text>Apple</Text>
    </Divider>
    <Text>Orange</Text>
    </Divider>
  </View>
);

Props

inset: boolean

Whether divider has a left inset

style: any
