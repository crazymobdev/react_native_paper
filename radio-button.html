<RadioButton />

Radio buttons allow the selection of a single option from a set Usage:

export default class MyComponent extends Component {
  state = {
    checked: 'firstOption',
  };

  render() {
    const { checked } = this.state;
    return (
      <View>
        <RadioButton
          value="firstOption"
          checked={checked === 'firstOption'}
          onPress={() => { this.setState({ checked: 'firstOption' }); }}
        />
        <RadioButton
          value="secondOption"
          checked={checked === 'secondOption'}
          onPress={() => { this.setState({ checked: 'secondOption' }); }}
        />
      </View>
    );
  }
}

Props

*checked: boolean

Whether radio is checked

disabled: boolean

Whether radio is disabled

onPress: Function

Function to execute on press

color: string

Custom color for radio

*theme: Theme
