RadioButton.Android

Radio buttons allow the selection a single option from a set. This component follows platform guidelines for Android.

Props

value (required)
Type: string

Value of the radio button

status
Type: 'checked' | 'unchecked'

Status of radio button.

disabled
Type: boolean

Whether radio is disabled.

onPress
Type: () => mixed

Function to execute on press.

uncheckedColor
Type: string

Custom color for unchecked radio.

color
Type: string

Custom color for radio.

theme
Type: Theme
